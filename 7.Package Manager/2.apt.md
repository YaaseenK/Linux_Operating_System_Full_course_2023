## APT-GET Command in Linux

**Basic Syntax:**
`sudo apt-get [options] [command] [package(s)]`

To search for a repo type the following command 
`sudo apt search [packageName]`

To install 
`sudo apt install [packageName]`

To uninstall 
`sudo apt remove [packageName]`
